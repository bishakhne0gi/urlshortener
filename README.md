

To get a a local copy on your system:

#Installation & setup

1. Clone the repository
  
    ```bash
     git clone https://github.com/bishakhne0gi/urlshortener.git
     ```
2. Change the directory
    ```bash
    cd urlshortener
    ```

2. Install the node modules

    ```bash
    npm install
    ```
3. Run the server

    ```bash
    npm start
    ```
